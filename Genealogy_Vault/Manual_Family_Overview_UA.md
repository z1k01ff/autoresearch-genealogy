# Огляд родинного дерева

Цей файл створений для наочного перегляду та ручного доповнення.

Важливо:
- Це не дослідницький файл.
- Цей файл зараз не повинен використовуватися як окреме джерело для пошуку.
- Якщо тут з'являться нові дані, їх потрібно окремо перевіряти перед внесенням у робочі дослідницькі нотатки.

## Як доповнювати

- Можна вписувати неточні або неповні дані.
- Невідомі місця позначайте як `невідомо`.
- Приблизні роки позначайте як `бл. 19xx`.
- Якщо не впевнені у факті, додавайте примітку `потрібно перевірити`.
